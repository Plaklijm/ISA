# ISA

ISA Project. 2.5D character controller inspired by ALSV4 and ZippyCharacterController

!W.I.P. not all features listed are implemented or are subject to change!

Features:
.Walk
.Run
.Sprint
.Crouch
.Mantling
.Rolling
.Push/Pull Boxes
.Walk on Spline
